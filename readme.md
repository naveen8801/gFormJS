# google-formjs

## Install

```bash
$ npm i -g google-formjs
```


## CLI

```
$ gformjs --help

  Description :
    Create google forms directly from terminal very easily

  Usage :
    Step 1 : gformjs
    Step 2 : Select form 2 options Create Form or Check Form Responses
    Step 3 : Type no. of question if creating a form

```
